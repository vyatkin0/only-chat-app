# only-chat
This is a text-based chat service that built on only-chat platform.

It is implemened using Elasticsearch and RabbitMQ stores and queues.


PREREQUISITES
-------------

- `node`: This requires Node 18 or greater.

INSTALL
-------

   ```sh
   $ npm install
   ```

TRY IT!
-------
 - Run the server

   ```sh
   $ # from this directory
   $ edit config.js
   $ npm run build
   $ npm run start
   ```
